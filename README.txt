# svg-app

* Tested on Google Chrome
* Includes one extra shape; Ellipse, try with 'e 120 200 60 30'

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
